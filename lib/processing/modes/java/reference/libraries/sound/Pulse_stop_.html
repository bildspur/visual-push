Class

Pulse

Name

stop()
Examples
import processing.sound.*;
Pulse pulse;

void setup() {
  size(640, 360);
  background(255);
    
  // Create the pulse oscillator
  pulse = new Pulse(this);
  pulse.play();
}      

void draw() {
}

void mousePressed() {
  pulse.stop();
}
Description Stops the Sine Oscillator generator.
Syntax
.stop()
Returnsvoid
